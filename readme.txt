== Description ==

Plug-N-Play!  Share MP3s, AVIs, ZIPs, DOCs and IMAGES via Your Admin Page. No Database Required. Plug n Play. Assumes WordPress Install is in ROOT. Visit <a href="http://grapefile.info" target="_blank">www.grapefile.info</a> BEFORE Every Upgrade or Risk Losing Your Files!

== Installation ==

*If your WordPress Install is NOT in the ROOT, but in a sub directory, vist the Grapefile site for Instructions.*

**What To Do**

1. Download the zip to your hardrive.
2. Upload to your 'Plugin' Directory via On-site Uploader (zip).
3. Click 'Activate Plugin'.
4. Thats it!!
5. User interface in Admin Sidebar.

== Frequently Asked Questions ==

= Will This Plugin Be Actively Developed? =

*Yes*. Development of this plugin will be **very active**.

= Will My Files Remain After A WordPress Core Update? =

*Yes*, the plugin is NOT effected by a Core Update.

= Will My Files Remain After A Grapefile Upgrade? =

Currently, your files 'may' be erased. This is not a for-sure thing. But, we recommend moving the storage location of the files upon activation of the plugin **(requires source code edit)**, and for Security.
